**For import to Replit — v1.0 MVP**

> Compiled from full design + strategy session.

## 1. WHO WILL USE IT (Personas & Context)

### Primary Persona — "Focus-First Knowledge Worker"
- **Age:** 24–42
- **Role:** Designer, developer, writer, researcher, creative professional
- **Behaviour:** Uses Pomodoro as a *ritual*, not a task manager
- **Need:** A timer that feels like a tool, not a distraction
- **Session:** 25 min focus + 5 min break (default)
- **Context:** Morning deep work, afternoon slump recovery, pre-meeting clarity

### Secondary Persona — "Minimalist Lifestyle User"
- **Aesthetic-conscious**, Apple ecosystem user
- Wants **one app per job**, not feature bloat
- Willing to pay for clean, purposeful design
- **Offline-only** usage; no account sync needed

### When & Where
- Desk work, home office, coffee shop
- **Not** during commute or meetings
- Morning + early afternoon peak usage

## 2. UX STYLE & SCREENS

### Onboarding (1 screen, 30 seconds)
1. **Welcome** — "Stay focused. One timer. Zero distractions."
2. **Pick your theme** — Midnight / Energy / Sunset / Break (live preview, radio buttons)
3. **Done** → lands on hero screen

> No tutorial. The UI teaches itself.

### Screen 1 — Hero / Dashboard ✓ (Designed in Figma)
Bento grid with 6 tiles:
- **Timer** (370×312, hero colour) — "25:00" + Start button + decorative ring
- **+5 min** (181×176, accent) — adds 5 minutes mid-session
- **Reset** (181×176, alert) — resets timer
- **Progress** (370×88, dark) — 8 session dots, 6 completed shown
- **Streak** (181×184, mirrors hero) — "7 day streak"
- **Ambient** (181×184, dark) — 3 chips (Rain / Lo-fi / Wind)

### Screen 2 — Session In Progress
- **Fullscreen timer** (large, centered, dominant)
- **Animated progress ring** (clockwise)
- **Single Pause button** (no reset during active session)
- **Session badge:** "Session 4 of 8"
- **Ambient pulse** indicator (if audio playing)
- **Gesture:** Swipe down to pause

### Screen 3 — Break Screen
- **Softer colour palette** (greens/blues, never red/orange)
- **5:00 countdown** to next focus
- **Two buttons:** "Next Focus" | "Skip Break"
- **Optional micro-copy:** "Great work! Take a breath."
- **Ambient continues** if user had it enabled

### Screen 4 — Daily Summary (v1.1, optional in v1.0)
- Week calendar — completed sessions per day
- Stats: "Total focus today: 2h 15m", "Longest streak: 7 days"
- "New Session" button → back to hero

### Button Sizes (all 8px multiples)
- **Start button:** 104 × 48
- **Action tiles:** 181 × 176
- **Ambient chips:** 48 × 48
- **Corner radii:** Tiles 24px, Phone shell 52px, Chips 14px

### Typography
- **Font:** Inter (Bold / Extra Bold / Medium only)
- **Timer number:** 96px Extra Bold
- **Tile labels:** 9px Bold, uppercase, 0.5 opacity
- **Body text:** 12–14px Medium

### Design System Rules
1. **Hero tile** = brand colour at full saturation
2. **Action tiles** = high-contrast accents (red/cream/blue/orange)

5. **All values on 8px grid**
6. **No gradients** — flat solid fills only

## 7. CORE LOGIC (Pseudocode)

### Timer Hook
```typescript
function useTimer(duration: number) {
  const [secondsLeft, setSecondsLeft] = useState(duration);
  const [isRunning, setIsRunning] = useState(false);
  
  useEffect(() => {
    if (!isRunning) return;
    const interval = setInterval(() => {
      setSecondsLeft(prev => {
        if (prev <= 1) {
          clearInterval(interval);
          handleSessionComplete();
          return 0;
        }
        return prev - 1;
      });
    }, 1000);
    return () => clearInterval(interval);
  }, [isRunning]);
  
  return { secondsLeft, isRunning, start, pause, reset, addTime };
}
```

### Session Persistence
```typescript
function saveSession(duration: number, completed: boolean) {
  const session = {
    id: crypto.randomUUID(),
    startedAt: new Date(Date.now() - duration * 1000).toISOString(),
    completedAt: new Date().toISOString(),
    duration,
    completed,
    label: null
  };
  
  const state = getStorageState();
  state.sessions.push(session);
  state.streak = recalculateStreak(state.sessions);
  setStorageState(state);
}
```

### Streak Calculation
```typescript
function recalculateStreak(sessions: Session[]): Streak {
  const today = new Date().toDateString();
  const dates = new Set(
    sessions
      .filter(s => s.completed)
      .map(s => new Date(s.completedAt).toDateString())
  );
  
  let current = 0;
  let date = new Date();
  while (dates.has(date.toDateString())) {
    current++;
    date.setDate(date.getDate() - 1);
  }
  
  return { current, longest: Math.max(current, getLongest()), lastSessionDate: today };
}
```

### Audio Player
```typescript
function useAudio(track: 'rain' | 'lofi' | 'wind' | null) {
  const audioRef = useRef<HTMLAudioElement | null>(null);
  
  useEffect(() => {
    if (!track) {
      audioRef.current?.pause();
      return;
    }
    audioRef.current = new Audio(`/audio/${track}-loop.mp3`);
    audioRef.current.loop = true;
    audioRef.current.volume = 0.5;
    audioRef.current.play();
    
    return () => audioRef.current?.pause();
  }, [track]);
}
```

## 9. AUDIO ASSETS (Source Guide)

### Where to Find Loops
1. **Freesound.org** — Free, CC-licensed, 30–60s clips
2. **Pixabay Music** — Royalty-free, no attribution required
3. **Zapsplat** — Free with account, professional quality
4. **Record your own:**
   - Rain: smartphone outside in a storm + EQ in Audacity
   - Lo-fi: GarageBand or Ableton (60s loop, low-key beat)
   - Wind: fan + EQ to remove low rumble

### File Specs
- **Format:** MP3, 320 kbps
- **Duration:** 30–60 seconds (loops seamlessly)
- **File size:** ~300–700 KB each
- **Total bundle:** ~1.5–2 MB (fine for web)

### Loop Quality Check
- [ ] No clicks at loop point (use fade in/out 50ms)
- [ ] Volume normalized to -14 LUFS
- [ ] Stereo, 44.1 kHz
- [ ] Test in browser — does it loop seamlessly?

## 10. RISK & MITIGATION

| Risk | Likelihood | Mitigation |
|------|-----------|-----------|
| Pomodoro market oversaturation | High | Differentiate via ambient + minimalism. Target designers, not generic productivity audience. |
| Users want cloud sync | Medium | Don't add it. If demand high in v1.0, defer to v2.0. Simplicity is the moat. |
| Audio bundle bloat | Low | Compress MP3s, lazy-load on first ambient toggle. |
| iOS App Store rejection | Low | No risky permissions (camera, location). Should pass easily. |
| Scope creep in v1.0 | Medium | **Lock the scope.** Only timer, sessions, audio, theme — no extras. |
